<h2>ğŸ§  About the Game</h2>

<p>
  <strong>BrainShuffle</strong> is a fun and interactive memory card shuffle game.
  The goal is simple: flip the cards, remember their positions, and match all
  pairs to win the game.
</p>

<p>
  The game includes smooth animations, sound effects, and a celebration animation
  when the player successfully matches all cards. It is designed to be lightweight,
  fast, and enjoyable for all age groups.
</p>
